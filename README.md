# Python Basic

This repository contains beginner-friendly Python learning materials and practice files. It is designed to help learners understand core Python concepts through notebooks and small scripts.

## Contents

- Capstone_Project.py - a starter capstone project file
- ConditionalStatement.ipynb - examples of conditional statements such as if, elif, and else
- DataStructures.ipynb - coverage of basic Python data structures
- Loops.ipynb - examples of for and while loops
- Pythondatatype.ipynb - explanations and examples of Python data types
- PythonString.ipynb - practice with Python strings and string methods

## How to Use

1. Install Python 3.x on your system.
2. Open the Jupyter notebooks with:
   ```bash
   jupyter notebook
   ```
3. Run the capstone script with:
   ```bash
   python Capstone_Project.py
   ```

## Purpose

This project is intended for learning and practicing Python basics, including:
- variables and data types
- conditionals
- loops
- data structures
- strings
- simple project structure
